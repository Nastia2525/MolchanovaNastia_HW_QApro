* “логічне AND” повертає значення останнього правдивого операнду. Number (5) та String(“qwerty” ) обидва true;

* об'єднання рядків (конкатенація) при додаванні;

* при відніманні String(‘10’ ) перетворюється на Number і результат віднімання буде Number (5). Він одного типу з Number (6), але математично вони не рівні;
6. true + false дасть 1
* Boolean (true) =1, Boolean (false) =0. 0+1=1;

8. '4' - 3 дасть 1
* без букв String('4' ) перетворюється на Number (4). 4-3=1;
9. '6' + 3 ** 0 дасть 61
* об’єднання рядків String('6') та Number (1) - 3 у нульвому ступені дорівнює 1;
10. 12 / '6' дасть 2
* арифметична дія перетворює String('6') на Number (6);

* “логічне AND” повертає значення останнього правдивого операнду. “логічне OR” виконується після “логічного AND” і дає true при правдивості одного з операндів;
